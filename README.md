# ExcaliDraw AI

A macOS desktop AI-powered diagram tool built with Tauri 2.x and Excalidraw.

## Features

- **AI Assistant Panel**: Natural language to diagram generation using Mermaid
- **Full Excalidraw Canvas**: Complete native Excalidraw functionality
- **Multiple LLM Providers**: DeepSeek, Qwen, GLM, OpenAI, Kimi
- **Liquid Glass UI**: macOS-style glassmorphism design
- **Local Storage**: API keys and settings stored locally only

## Tech Stack

- Tauri 2.x
- React 18 + TypeScript
- Vite
- Zustand
- @excalidraw/excalidraw
- @excalidraw/mermaid-to-excalidraw

## Development

```bash
npm install
npm run tauri dev
```

## Build

```bash
npm run tauri build
```
