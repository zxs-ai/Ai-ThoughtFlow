### 🌟 什么是 Ai ThoughtFlow Pro？

**Ai ThoughtFlow Pro** 是一款专为**教育、创意、与知识整理**设计的跨平台 AI 智能绘图工具。它不仅仅是一个转换器，它是一个能够理解你意图的「图形设计师」。

通过将 **Excalidraw** 的手绘美学与最新的 **LLM 大模型** 逻辑相结合，Ai ThoughtFlow Pro 能将一段凌乱的文字，瞬间转化为充满温度、层次分明、且可交互编辑的专业图表。

---

### 🔥 核心五大能力

#### 1. ✨ 智能自动选型 (Intelligent Auto-Mode)
**不再纠结选什么图。** 
只需输入你的想法，AI 会自动分析语境，在思维导图、流程图、双气泡对比图、因果流程图等 **9 种** 专业模型中自动匹配最优结构。

#### 2. 🧠 深度上下文对话 (Conversational Editing)
**像聊天一样改图。**
觉得节点太少？想加个分支？还是想换种风格？
直接在对话框输入：「帮我再加两个关于环境影响的分支」或「把这个改成横向布局」。AI 会基于当前图表进行逻辑扩展，无需从头开始。

#### 3. 🎨 卡通教育美学 (Aesthetic Design)
**告别冷冰冰的工业风。**
内置定制的高对比度卡通配色方案与手绘质感，让复杂的逻辑变得生动有趣。非常适合教师课件制作、学生知识梳理以及产品经理的头脑风暴。

#### 4. 📐 v4 递归防重叠引擎 (Anti-Overlap Layout)
**极致排版，强迫症福音。**
采用自研的递归几何布局算法，自动计算末端节点的安全间距。无论分支多复杂，节点永不重叠，始终保持画面整洁。

#### 5. 🧩 100% 可交互编辑 (Native Interaction)
**生成的不是死图，是活的。**
所有生成的元素（矩形、线条、文字）均为原生 Excalidraw 元素，支持手动拖拽、调整颜色、二次编辑，满足你对每一个像素的控制欲。

### 🛠️ 技术栈

*   **UI 框架**: React 18 + TypeScript (液态玻璃视觉系统)
*   **跨平台**: Tauri 2.0 (Rust 驱动，极速启动，超低内存)
*   **布局引擎**: 自研 v4 递归放射状/树形/因果流布局算法
*   **绘图核心**: Excalidraw SDK
